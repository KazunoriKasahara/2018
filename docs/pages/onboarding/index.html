Onboarding

Please complete the following before the first day of class.

  1. Sign up for our Slack channel. You should have received an invite. If not email the instructors.
  2. Sign up for a GitHub
  3. Fill out course survey using the course. The should have received an email with the link. A link is also posted on the Slack channel.
  4. Fill out the survey with office hour preferences. See the email sent to the class or the Slack channel for a link.
  5. Sign up for a meeting with Jeff Arnold. See the email sent to the class or the Slack channel for a link.
  6. Download R and RStudio Desktop (Open Source Edition). You should be able to use defaults for all settings.
  7. Download Git. You should be able to use defaults for all settings.
  8. Read the current state of the syllabus. Open an issue on GitHub. In the issue ask at least one question and/or make one suggestion about the content of the website. This task is both to ensure you have signed up for GitHub and also to provide me with information about how to improve the site.
  9. Sign into Slack and introduce yourself in a comment. We will also introduce ourselves in class, but this task ensures that you have access to Slack and also introduces yourself.

Done! See you in class on Friday! Please get started on any reading. You may also consider starting your R education with the DataCamp course Introduction to the Tidyverse.
