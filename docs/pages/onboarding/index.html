Onboarding

Please complete the following before the first day of class.

  1. Sign up for our Slack channel. You should have received an invite. If not email the instructors.
  2. Sign up for a GitHub
  3. Fill out course survey using the course. The should have received an email with the link. A link is also posted on the Slack channel.
  4. Download R and RStudio Desktop (Open Source Edition). You should be able to use defaults for all settings.
  5. Download Git. You should be able to use defaults for all settings.

Congrats! See you in class. Get started on any ready and.
